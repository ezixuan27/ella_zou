# Zixuan (Ella) Zou — Personal Portfolio

My personal portfolio website, built with React + Vite and styled with Tailwind CSS.

## Features

- Animated loading screen
- Home, Projects, and Experience pages with smooth page transitions
- Scroll-driven reveal animations
- Interactive polaroid photo gallery
- Animated timeline for education and experience
- Mobile-responsive design with a slide-in menu
- Contact section

## Tech Stack

- **React** + **Vite**
- **Tailwind CSS**
- **Lottie** (loading animation)

## Running Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
